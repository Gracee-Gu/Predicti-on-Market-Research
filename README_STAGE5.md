# Stage 5 — Manuscript Integration, Evidence Traceability, and Reproducible Release

Stage 5 converts the completed Stage 1–4 research assets into an auditable manuscript package. It does not strengthen claims beyond the Stage 4 `claim_status`. The workflow assembles a paper draft, builds a claim–evidence ledger, generates a reproducibility manifest, checks citations and required sections, and prepares a release checklist.

Start with `docs/runbook_stage5.md`.
