# Stage 4 — Confirmatory Analysis, Robustness, and Reporting

Stage 4 converts the Stage 3 adjudicated article–market dataset into auditable diagnostics, hypothesis tests, robustness checks, tables, figures, and a paper-ready results scaffold. Start with `docs/runbook_stage4.md`.

The workflow blocks confirmatory claims when source-type variation, market-quality coverage, sample size, or provenance requirements are not met.
