# Final Paper

- `manuscript.html` — source (plain HTML/CSS, two-column print layout, no LaTeX dependency)
- `manuscript.pdf` — rendered PDF (see repository root `README.md` for the render command)
- `figures/` — all figures referenced in the manuscript, generated by `scripts/make_figures.py`
  from `data/analysis/article_market_dataset.csv` and `outputs/verification/`
- `data_and_code_availability.md` — data and code availability statement
- `explainer.html` — a plain-language, visual companion to the manuscript, summarizing the design,
  key figures, and two case exhibits for a reader who wants the argument at a glance rather than
  the full paper
