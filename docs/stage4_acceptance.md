# Stage 4 Acceptance Criteria

Implementation passes when:

- installation adds files without silently overwriting repository files;
- input discovery supports the current Stage 3 filename and documented aliases;
- schema and value validation run before analysis;
- claim gates are machine-readable;
- descriptive tables, effect estimates, multiplicity adjustment, and robustness registry are generated;
- unavailable analyses are explicitly logged;
- tests pass.

Empirical completion requires a corpus that passes claim gates and a provenance record establishing human adjudication. A successful script run alone is not empirical completion.
