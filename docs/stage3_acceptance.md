# Stage 3 Acceptance Criteria

Stage 3 passes implementation when: sampling is deterministic; coding schema validates; blind duplicate assignments are produced; reliability supports nominal and ordinal fields; adjudication preserves coder provenance; analytic joins reject duplicate pair IDs; and outputs distinguish historical price/trade evidence from prospective order-book evidence.

Empirical completion additionally requires a populated coding sheet, documented adjudication, alpha reporting, and a nonempty analytic dataset. Code delivery alone is therefore an implementation pass, not an empirical-results pass.
