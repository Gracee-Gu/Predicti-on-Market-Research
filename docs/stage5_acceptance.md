# Stage 5 Acceptance Criteria

Implementation passes when:

- installation does not silently overwrite repository files;
- Stage 4 status and required outputs are discovered through documented aliases;
- a manuscript with all required sections is generated;
- every empirical results paragraph includes the Stage 4 interpretation boundary;
- a machine-readable claim–evidence ledger is generated;
- a reproducibility manifest records file paths and SHA-256 hashes;
- release auditing checks secrets, raw copyrighted text indicators, missing files, and causal-language violations;
- tests pass.

Research completion is conditional on the Stage 4 empirical status. Stage 5 cannot upgrade `exploratory_only` or `blocked` evidence to confirmatory evidence.
